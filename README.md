# GR-EN-A-DE: A Novel Graph-Based Model for online  Extremist Narrative Analysis

This repository contains the code associated with the paper titled "GR-EN-A-DE: A Novel Graph-Based Model for online  Extremist Narrative Analysis" currently under review.


## Overview

The purpose of this repository is to support the reproducibility of the experiments and results reported in the paper. To preserve the anonymity of the authors during the review process, identifying information has been removed or generalized.

## Usage

The code is organized to allow straightforward execution of the experiments described in the paper. Instructions and parameters are provided within the scripts where necessary.


## Requirements
The dependencies are specified in the file `requirements.txt`.
