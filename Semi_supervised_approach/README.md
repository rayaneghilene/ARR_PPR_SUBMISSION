# Semi-Supervised Graph Learning Framework:

This repository provides utilities for dataset preprocessing, text embeddings, graph construction, and training pipelines for node classification and link prediction (not explored yet). It supports the Multilingual EN, Toxigen and FRENK Dataset

## Installation

We recommend creating a virtual environment (optional):
```bash
python3 -m venv myenv
source myenv/bin/activate 
```
To install the requirements run the following command: 

```bash
pip install -r requirements.txt
```

and then run:

```bash
python main.py
```
